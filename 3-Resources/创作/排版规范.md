# 二、基础格式要求

## 1. 段落结构

• 每段之间空一行，不要缩进，不要一整坨堆在一起。

• 一段控制在 3~5 行以内，超过就拆。别嫌麻烦，眼睛比你还累。

  

## 2. 空格使用（重点）

✅ 以下场景 必须加空格：

• 中文和英文之间：

• 错误：在LeanCloud上开发很方便

• 正确：在 LeanCloud 上开发很方便

  

• 中文和数字之间：

• 错误：上线时间是2025年

• 正确：上线时间是 2025 年

  

• 数字和单位之间：

• 正确：10 Mbps、5 GHz

• 错误：10Mbps、5GHz

  

• 中文和链接、加粗、斜体等格式之间：

• 正确：请 点击这里 参与活动

• 错误：请点击这里参与活动

  

• 中文和破折号之间：

• 正确：这是重点 —— 别写错了

• 错误：这是重点——别写错了

  

❌ 以下场景 不要加空格：

• 数字 + 特殊单位（℃、%、°）：

• 正确：今天是 28℃，转化率为 15%。

• 错误：今天是 28 ℃，转化率为 15 %。

  

# 三、标点符号

• 中文语境下，用全角标点，不要夹英文逗号、句号、引号。

• 正确：「这样」，「那样」。

• 错误：“这样”, ‘那样’。




  

• 不要重复标点（强调可以重复）：

• 正确：你真的来了！

• 错误：你真的来了！！！！！！

  

• 破折号请前后空一格：

• 正确：这是重点 —— 必看！

• 错误：这是重点——必看！

• 省略号一格两点，写成……，不要…… 或 。。。。

  

# 四、名词大小写与术语规范

• 专有名词注意大小写：

• 正确：使用 GitHub 登录

• 错误：使用 github 登录

  

• 产品名、品牌名按官方格式写：

• 正确：豆瓣FM、Magic Eden、Galxe  
• 正确：豆瓣 FM、Magic Eden、Galxe

• 错误：豆瓣 fm、magic eden、galxe

  

• 数字请用半角字符：

• 正确：我花了 1000 元买了个显卡

• 错误：我花了 １０００ 元买了个显卡

  

• 不使用奇怪缩写：

• 错误：前端 FED 需要懂 Js 和 H5

• 正确：前端需要懂 JavaScript 和 HTML5

  

# 五、链接处理

• 文内链接使用加粗，提升点击率：

• 正确：**[http://x.com/Xuegaogx](http://x.com/Xuegaogx)**



• 错误：[http://x.com/Xuegaogx](http://x.com/Xuegaogx)
  

• 链接前后要空格，不要贴得太紧：

• 正确：参与活动请点击 [这里](http://x.com/Xuegaogx)。

• 错误：参与活动请点击[这里](http://x.com/Xuegaogx)

。

  

# 六、写作风格建议（进阶）

• 如无必要，勿增实体。

• 拆长句，多断句。能用两个句子完成的，就不要非要一个句号撑到最后。

• 每个论点后都尽量举个例子，别光说理不贴地。

  

# 七、特别强调

• 简体中文推荐使用直角引号：「」而非“”。

• 中英文混排时，整句英文内容请使用英文半角标点。

• 正确：「Stay hungry, stay foolish.」  
• 正确：“保持饥饿，保持愚蠢。”

• 错误：「Stay hungry，stay foolish。」  
• 错误：“保持饥饿，保持愚蠢。”
