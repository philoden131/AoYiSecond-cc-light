大环境越好，大家越乐观，越要聊增长
大环境越差，大家越悲观，越要聊盈利

要强调人的重要性：
1. CEO：的眼界水平 等于 公司上线

3. 所做内容：项目从成立到现在，做了什么，什么速度做的，经历了什么，做了哪些调整。如何在创业过程中调整节奏的

# MyWear BP

企业亮点：
1. 项目是什么：
2. 核心数据和发展情况
3. 创始人

一句话：
特点描述+成功项目（是已经被压缩的概念了

中国的/东南亚的/年轻人的/更便宜的

Youtube 抖音 微信 AirPods 等
可能不准确，但是能一句话说清楚 主要是做什么

坑点：描述不能过于窄 / 对标项目需要大，牛（投资人认为的好项目是 能赚到钱的项目

- 摘要
	- 项目
		- Mywear：做每个人都能拥有的私人穿搭管家
	- 数据（成绩？
		- 
	- 创始人/CEO
		- 标签，连续创业者，有经验，赚过钱，拿过投资
		- 现在想成为时尚行业教父


- 团队
	- 早期项目着重展示CEO
	- 其他成员展示能够独当一面  / 团队完整性
	- 主要写经历：
		- 一级经历：赚过钱的连续创业者/创业成功的精力/相关的成功精力/大公司高管/拿过知名机构的投资（最好是帮投资机构赚到过钱的
		- 二级精力：行业专家/ 失败的创业经历 / 名校毕业 
		- 三级经历：从业年限 / 不太相关的成功经验（电竞冠军？  成功的人更容易再次成功，虽然和项目没关系【太小的成功不要提，**太空泛的词不要提**
	- 举例：李自然联合创始人&CEO
		- 前xxxCEO,海外第一的xx项目,MAUxxxxx
		- 前xxxCEO,某北美项目,年营收xxxxxx
		- 前xxxCEO,东南亚xx项目,获得xx资本投资
		- 《李自然说》主讲人,创投圈第一的长视频自媒体
		- 有经验、赚过钱、擅长流量运营、有做内容的激情、想赢 


- 项目背景（告诉投资人了解行业背景 - 对行业的理解是否靠谱
	- 需求：
		- 行业发展到什么阶段了，产生了什么需求
	- 规模：
		- 宏观数据 - 自上而下 （整体规模 - 我们占有率 能做到多少等
		- 微观数据 - 自下而上 （单店模型 - 放大规模 - 总规模
		- 最好终局规模是能达到上市标准的（10亿？
		- 初创团队数据准不准其实不太重要，但是分析思路要准确
		- 写的东西不要过于科普向
		- 例子：
			- 每个行业都有一个长视频KOL空间，年营收10亿（讲清楚具体怎么算
			- B站单月上传学习类视频28000+
- 产品：
	- 让人直观的感受产品是什么样子的
	- 有哪些内容
- 竞争优势：
	- 哪自己的优势和别人做对比
	- 也要承认事实，认可竞品的优点
	- 拼的也是市场的认知 对背景的了解程度
- 竞品分析：
	- 要对竞品研究的很透
	- 投资人会重点询问
	- 没有对手就是没有市场 / 如果细分领域没有竞品 就需要扩大市场 例如：果汁领域 扩展到饮料领域
	- 找的竞品不要太弱 - 太弱说明市场不行 / 投资人会自行去对比
	- 不同纬度的竞品 从不同纬度去对比
- 商业模式：
	- 商业模式太多 不太好。
- 路线图：
		- 着重展现发展速度（不是写产品进度  要主要展现**增长**  强调项目**整体进展**的路线，而不是产品版本号，写什么时候进入什么阶段
		- 未来不能写的太过，要站得住脚 ，不能太务虚/ 展望未来不要太远，因为市场环境变化很快
		- 例子：
			- yyyy.mm.dd 项目立项
			- yyyy.mm.dd  demo上线
			- yyyy.mm.dd  推出xx功能，日活增量xx 
			- yyyy.mm.dd  项目开始盈利
			- yyyy.mm.dd  达到xx收入（合理
- 融资需求
	- 报价（对自己公司估值有个合理的预估区间 然后出让多少股份  要合理
		- 例子：
			- 融资200w人民币，出让10%
			- 团队已自投100w（让投资人放心，加分项
	- 股权结构（要股权架构健康（CEO需要绝对控股
		- 例子：
			- CEO：80%。
			- 合伙人A：10%
			- ESOP：10%
	- 后续融资计划：（告诉投资人，让他放心，给他找接盘侠
		- 例子：6-8个月后启动A轮融资 等
				
	- 资金使用计划：
			- （要写至少10个月 12-18个月的钱左右  （不能太长 不能太短（表达创始人的创业思路 看把钱花在营销上，还是把钱花在产品开发上
			- 重要的是体现创始人的思路
				- 例子：（基本上拍脑袋定的 早期项目 相对合理就行了
					- 场地设备：
					- 人员工资：
					- 推广运营：
- 联系方式
	- 创始人自己：
		- 手机
		- 微信
