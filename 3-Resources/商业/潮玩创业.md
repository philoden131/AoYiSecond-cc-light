**调整策略说明：**

1. **保持原有层级**：结构回归到符合业务逻辑的 **基础 -> 物理 -> 美学 -> 场景** 四大标准分类。
    
2. **颜色编码 (Color Coding)**：我将节点的底色修改为三种状态，直观区分数据的消费对象：
    
    - 🟦 **蓝色 (Hybrid - 双向共识)**：AI 能识别且用户高频关注（如：季节、主色、子品类）。
        
    - 🟪 **紫色 (AI Only - 隐性计算)**：用户不直接看，但 AI 需要据此计算风格（如：结构感、量感）。
        
    - 🟧 **橙色 (Human - 显性感知)**：AI 难以精准识别或专供用户阅读的信息（如：精确材质、设计故事、搭配建议）。
