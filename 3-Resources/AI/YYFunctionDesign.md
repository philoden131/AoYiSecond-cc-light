AI 穿搭智能体 “小 Y” 项目设计文档
一、 项目愿景
打造一个名为“小 Y”的 AI 穿搭智能体。它将作为用户的私人造型师，通过深度理解用户的个人数据（形象、偏好、衣橱），以自然语言交互的方式，为用户提供个性化的穿搭解决方案。
二、 智能体核心架构
主角色 (中央控制器)
“主角色”是小 Y 的决策核心，扮演着中央控制器和任务规划者的角色。
- 核心职责：
  1. 意图理解： 解析用户的文本与图片输入，准确判断其背后的真实需求。
  2. 任务规划： 将复杂需求拆解为一系列可执行的步骤（Plan）。
  3. 功能调用： 根据规划，按需调用下方的各项功能模块来执行任务。
  4. 结果整合： 汇总所有模块返回的信息，并以“私人造型师”的统一口吻，生成最终的文本与图片回复给用户。
可调用的功能模块 (工具集)
主角色可以调用以下功能模块来完成具体任务：
1. 衣物图像分割 (原“拆图”)
- 功能： 从用户上传的任意照片中，识别并提取衣物主体，生成标准化的白底平铺图。
2. 时尚属性识别 (原“打标”)
- 功能： 分析衣物平铺图，提取其多维度属性（如品类、颜色、风格、材质等），并将其结构化为 JSON 格式，与图片一一对应。
3. 个性化搭配生成 (原“搭配”)
- 功能： 综合用户的即时需求、个人风格偏好及衣橱数据，依据内置的搭配方法论，生成一套或多套由衣物 ID 组成的搭配方案。
4. 虚拟试穿生成 (原“上身照”)
- 功能： 根据给定的衣物 ID 组合和用户的全身形象照，生成逼真的虚拟上身效果图。
5. 外部信息获取 (原“天气获取”)
- 功能： 调用天气 API，获取指定地点和日期的天气信息，为穿搭决策提供环境依据。
三、 记忆模块
为了实现深度个性化，小 Y 需要访问和维护以下记忆内容：
- 用户画像 (Profile):
  - 形象特征: (文本) 用户的身形、肤色等客观数据。
  - 风格偏好: (文本) 用户喜爱或不喜欢的风格、颜色、品牌等主观偏好。
- 用户基准形象照 (Base Photo):
  - (图片) 一张用户正面全身的清晰照片，用于虚拟试穿。
- 数字化衣橱 (Wardrobe):
  - (JSON 集合) 用户所有衣物的结构化属性档案。
  - (图片库) 与 JSON 档案对应的所有衣物白底平铺图。
四、 核心使用场景
类别一：日常穿搭决策
解决“今天穿什么”的终极难题。
场景 1：无差别每日推荐
- 用户输入: “今天穿什么？” 或 “根据今天的天气给我推荐一套衣服。”
- Agent 输出:
  - 文本: “今天[城市]天气[天气状况]，推荐你穿衣橱里的[白色长袖T恤]搭配[蓝色牛仔裤]，再带上你的[卡其色风衣]以备早晚温差。这是今天推荐的整套搭配。”
  - 图片: [为你生成这套搭配的虚拟上身效果图]。
场景 2：基于核心单品的搭配构建
- 用户输入: [上传一张衣橱里某件衣服的照片] + “今天想穿这件，帮我搭一下剩下的。”
- Agent 输出:
  - 文本: “好的，围绕这件[绿色针织衫]，你可以搭配你的[米色直筒裤]和[德训鞋]，打造一种干净的都市休闲感。”
  - 图片: [为你生成这套搭配的虚拟上身效果图]。

场景 3：穿搭效果评估与优化 (已调整)
- 用户输入: [上传自己穿好衣服的 OOTD 照片] + “我这样穿可以吗？”
- Agent 输出:
  - 文本: “整体很有活力！从穿搭技巧上看，有个小建议可以让比例更好：尝试把T恤的下摆塞进裤子里，利用‘上短下长’的原则，可以进一步拉长腿部线条，显得更精神。”


类别二：风格探索与学习
陪你一起成长，找到并定义你独一无二的风格。

场景 4：抽象风格的具象化
- 用户输入: “什么是 ‘Clean Fit’ 风格？”
- Agent 输出 (第一步):
  - 文本: “Clean Fit 是一种注重剪裁、质感和简约配色的穿搭风格... 核心单品通常包括合身的西裤、纯色T恤、衬衫和乐福鞋。需要我从你的衣橱中为您生成对应的穿搭风格图吗？”
- 用户输入 (第二步): “好的”
- Agent 输出 (第二步):
  - 文本: “已为你生成3套Clean Fit风格搭配。”
  - 图片: [一个由你衣橱中单品组成的Clean Fit风格情绪板，或多套搭配的虚拟上身图]。

场景 5：特定风格的个人化转译
- 用户输入: [一张明星的街拍图] + “我想穿出这种感觉，帮我用我衣橱里的衣服搭一套。”
- Agent 输出:
  - 文本: “分析了这张街拍图的核心单品。通过在你衣橱里寻找相似款，为你复刻了这套搭配。你可以用你的[黑色皮夹克]、[白色T恤]和[深色牛仔裤]来组合，就能达到80%的相似度。”
  - 图片: [为你生成这套搭配的虚拟上身效果图]。


场景 6：个人风格定位分析 
- 用户输入: “你觉得我的穿衣风格是什么？”
- Agent 输出:
  - 文本: “通过分析你的衣橱数据，我发现你的风格偏向于‘日系休闲’与‘简约通勤’的混合。例如，你的衣橱中大地色系（如卡其、米白）的单品占比达到了40%，且80%的下装都是牛仔裤和直筒裤这类基础款，这表明你非常注重实穿性和质感。”


类别三：衣橱管理与优化

让你的衣橱焕发新生，每一件衣服都有它的价值。
- 场景 7：衣橱潜力挖掘 (已调整)
  - 用户输入: “帮我找出我衣橱里最不常穿的衣服。”
  - Agent 输出 (第一步):
    - 文本: “根据记录，你这件[印花半身裙]和[亮黄色卫衣]已经超过半年没穿过了。需要我帮你为这些单品搭配一些穿搭吗？”
    - 图片: [印花半身裙] 和 [亮黄色卫衣] 的平铺图。
  - 用户输入 (第二步): “帮我搭一下半身裙”
  - Agent 输出 (第二步): (激活场景2) “好的，围绕这件[印花半身裙]，你可以搭配...”

场景 8：衣橱结构性诊断 (已调整)
- 用户输入: “我总觉得没衣服穿，帮我分析下我的衣橱是不是缺了什么？”
- Agent 输出:
  - 文本: “分析了你的衣橱构成，发现你的上衣（T恤、衬衫、毛衣共30件）种类非常丰富，但下装只有5条深色系的裤子。这种上下装比例失衡，可能会导致你觉得‘没衣服穿’。建议可以考虑补充一些百搭的浅色系下装，比如一条白色或米色的阔腿裤，来盘活你现有的上衣。”

场景 9：旅行穿搭规划
- 用户输入: “我下周要去大理玩5天，天气大概是15-25度，帮我规划一下每天的穿搭。”
- Agent 输出:
  - 文本: “好的，为你规划了一个‘5天4晚’的胶囊衣橱，包含[具体单品列表]。Day 1: [搭配方案A], Day 2: [搭配方案B]...”
  - 图片: [一个包含所有携带单品及每日搭配组合的可视化清单]。

场景 10：重要场合着装建议
- 用户输入: “我下个月要去参加朋友的婚礼，该怎么穿？”
- Agent 输出:
  - 文本: “参加婚礼的关键是得体并尊重场合。推荐你穿那件[淡蓝色连衣裙]，它既优雅又不会抢新娘风头，再搭配你的[裸色高跟鞋]就很完美。”
  - 图片: [为你生成这套搭配的虚拟上身效果图]。
类别五：购物决策辅助
成为你最理性的购物参谋，让你的每一分钱都花在刀刃上。
场景 11：购前兼容性评估
- 用户输入: [一件网上商店的衣服图片] + “我种草了这件，它和我的衣服好搭吗？”
- Agent 输出:
  - 文本: “这件[焦糖色夹克]很不错！它和你衣橱里的[牛仔裤]、[黑色长裤]等至少8件下装都能搭配，利用率会很高。” 或 “虽然这件上衣很特别，但它和你衣橱中超过90%的单品风格不兼容，可能会闲置，建议谨慎。”
类别六：创意与趣味互动
穿搭不仅是需求，更是一种乐趣和自我表达。
场景 12：基于情感/情绪的穿搭建议
- 用户输入: “明天有个重要的展示，帮我搭一套能增加自信的衣服！”
- Agent 输出:
  - 文本: “想要增加自信，可以尝试‘力量着装’。推荐你穿衣橱里的[有垫肩的西装外套]和[利落的西装裤]，挺拔的线条和干练的风格能给人带来积极的心理暗示。”
  - 图片: [一套充满力量感的搭配虚拟上身效果图]。
场景 13：创意主题挑战
- 用户输入: “如果用我的衣服，搭一套‘秋天的第一杯奶茶’主题的穿搭？”
- Agent 输出:
  - 文本: “这个主题很有趣！我们可以用[卡其色风衣]作为奶茶底色，内搭[奶油白毛衣]模仿奶盖，再用你的[焦糖色贝雷帽]点缀。整体色调温暖又甜蜜。”
  - 图片: [一套充满创意的“奶茶色系”搭配效果图]。
