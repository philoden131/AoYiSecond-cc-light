# Alta 竞品调研报告

> 调研时间：2025-02-23
> 调研目的：为 MyWear AI 加速器申请提供竞品分析参考

---

## 一、公司概况

| 项目 | 内容 |
|------|------|
| **公司名称** | Alta (Alta Daily) |
| **产品定位** | Your personal AI stylist that truly gets you（真正了解你的个人AI造型师） |
| **成立时间** | 约2023-2024年（App Store上线） |
| **产品形态** | iOS + Android 双平台移动应用 + 网页端 |
| **社交媒体** | Instagram: @alta / TikTok: @altadaily |

---

## 二、创始人愿景

Alta 的核心愿景是：**让每个人都能轻松获得专业的时尚造型建议**。

从产品定位来看，创始团队希望解决的问题是：
- 日常穿搭决策耗时
- 现有衣橱利用率低
- 缺乏专业造型指导

→ 与 MyWear 解决的问题高度相似，形成直接竞争关系。

---

## 三、产品功能矩阵

| 功能 | Alta | MyWear（对标） |
|------|------|----------------|
| AI对话造型 | ✅ 通过对话告知计划，AI推荐穿搭 | ✅ 已实现 |
| 衣橱管理 | ✅ 整合现有衣物，一站式查看 | ✅ 已实现（拍照+AI自动打标） |
| 虚拟试穿 | ✅ AI生成avatar试穿新单品 | ❌ 未实现 |
| 场景化穿搭 | ✅ 工作、约会、旅行、早午餐等 | ✅ 已实现 |
| 天气穿搭 | ✅ 根据天气推荐 | ❓ 需确认 |
| 电商整合 | ✅ 与Poshmark合作二手奢侈品 | ❌ 规划中 |

**关键差异点**：
- Alta 侧重于**虚拟试穿**和**电商导流**（已与Poshmark、TaskRabbit合作）
- MyWear 侧重于**现有衣橱激活**（用户拍自己的衣服，AI打标）

---

## 四、商业模式

### 4.1 变现路径

Alta 的商业模式体现了典型的 **AI穿搭产品变现公式**：

```
早期：订阅制（月费/年费）验证PMF
    ↓
中期：电商分销分成（与Poshmark等平台合作）
    ↓
后期：品牌广告（基于用户穿搭数据的精准投放）
```

### 4.2 合作伙伴生态

- **Poshmark**：二手奢侈品包包购物整合
- **TaskRabbit**：用户可预约摄影师上门拍衣橱照片（解决冷启动录入难题）

→ 这两个合作揭示了 Alta 的核心挑战：**用户衣橱数据的冷启动**。TaskRabbit的合作说明 Alta 发现让用户自己拍照上传是一个摩擦点。

## 八、对 MyWear 的启示

### 8.1 竞争启示

Alta 已占据先发优势，但存在以下机会点：

| 机会点 | Alta现状 | MyWear可切入 |
|--------|----------|--------------|
| 衣橱录入体验 | 需预约TaskRabbit上门拍照 | 用户自主拍照+AI自动打标（已解决） |
| 亚洲市场 | 主要面向欧美 | 可专注日韩市场 |
| 定价 | 偏高（推测） | 可用更低价格切入 |

### 8.2 差异化建议

1. **强化AI打标能力**：这是 Alta 的痛点，MyWear 若能做到"拍照即入库"将形成显著优势
2. **本地化**：专注日韩市场（高时尚消费 + 与中国文化接近）
3. **轻量化起步**：先聚焦"现有衣橱激活"而非"虚拟试穿"

### 8.3 风险提醒

- Alta 已与 Poshmark 等平台建立电商合作关系，形成生态壁垒
- 虚拟试穿是 Alta 的核心差异化，MyWear 若不跟进可能处于被动
