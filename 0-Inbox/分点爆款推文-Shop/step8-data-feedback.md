# Step 8：数据反馈学习（发布后）

当 Roland 反馈推文数据时（查看/点赞/转发），执行以下操作：

1. 将数据追加到 `01-内容生产/方法论/分点改写爆款实验.md` 的测试记录中
2. 根据表现分级：
   - 查看 ≥10,000 或 点赞 ≥100 → 加入 `reference-cases.md` 的高表现案例
   - 查看 <2,000 且 点赞 <20 → 加入 `reference-cases.md` 的失败案例，分析原因
3. 每积累 3 条新案例，回顾一次 `core-rules.md`，看是否需要新增或修正规则

触发方式：Roland 说"数据更新"或提供推文数据时自动执行。
