# 软饭文学子技能

**触发词**：软饭文学

## 定位

双向改写：
- 包养/偷情素材 → 改写成"吃软饭上门女婿"的故事
- 吃软饭素材 → 改写成"包养/偷情"的故事

## 素材适配性判断（改写前必做）

在改写之前，必须先判断素材是否适合改成软饭文学。

### 判断步骤

**Step 1：识别原文的权力结构**
- 谁有钱？谁没钱？
- 谁养谁？谁依赖谁？
- 矛盾点是什么？

**Step 2：判断是否适合改写**

✅ **适合改写的素材类型**：
- 包养关系素材（有明确的养与被养）
- 出轨故事（可以转换关系）
- 吃软饭素材（本身就是软饭素材，可以反向改成包养）
- 权力结构清晰的素材（富养穷，或者穷依赖富）

❌ **不适合改写的素材类型**：
- 富家女被穷男方家要彩礼（权力结构相反）
- 富家女被穷男方家嫌弃（权力结构相反）
- 双方经济地位相当的素材（没有明确的养与被养）

**Step 3：如果不适合，提供替代方案**

如果素材不适合改成软饭文学，告诉用户：

```
这个素材不太适合改成软饭文学，因为：
[说明原因]

可以考虑：
- 方案A：改成"富家男被穷女方家要彩礼"（性别对调，但不是软饭文学）
- 方案B：强行改写，但需要修改核心矛盾点（比如把"要我家出8.8万"改成"岳母嫌我家穷，当初只收了8.8万"）
- 方案C：换一个更适合的素材
```

### 典型案例

**案例1：富家女被穷男方家要彩礼**
- 原文权力结构：富家女（有学区房+车+现金）vs 穷男方家（要8.8万彩礼）
- 软饭文学权力结构：穷男（被养）vs 富女方家（养他）
- 判断：❌ 不适合，因为权力结构相反
- 问题：如果我是穷男，岳母不应该要我家出8.8万彩礼（我家没钱）
- 替代方案：改成"岳母嫌我家穷，当初只收了8.8万彩礼，现在还经常念叨"

**案例2：包养人妻素材**
- 原文权力结构：我（有钱）包养情人（没钱）
- 软饭文学权力结构：我（没钱）被老婆养（有钱）
- 判断：✅ 适合，权力结构可以直接转换
- 改写方向：我从"包养者"变成"被养者"，情人从"被包养"变成"老婆（养我）"

## 核心规则

### 规则1："我"始终是男性视角
无论原文是男性还是女性视角，改写后"我"都是男性。

### 规则2：必须保留原文的具体情节细节
原文出现的数字、场景、对话、动作，改写时必须保留，只替换角色关系。
- ✅ 原文"68块钱麻辣烫" → 改写后保留"68块钱麻辣烫"
- ✅ 原文"西红柿炒鸡蛋和青椒肉丝" → 改写后保留
- ❌ 原文有具体情节，改写时自己编新内容

### 规则3：人物设定必须匹配
改写后的人物经济状况、身份、能力必须和新角色匹配。
- 原文"女大学生在化妆品柜台兼职"（穷）→ 改写后不能保留，因为老婆是富婆
- 原文"租的房子太破"（穷）→ 改写后不能保留，因为老婆有房有车
- 原文"男的穿裙摆"→ 改写后必须改成男性服装
- 原文"我怀孕了"→ 如果"我"是男性，必须改成"她怀孕了"

### 规则4：事实逻辑核查（改写完必须过一遍）
改写完成后，逐句检查以下逻辑：

| 检查项 | 问自己 |
|--------|--------|
| 性别逻辑 | "我"是男的，这句话男的能做吗？（怀孕、穿裙子 = 不能） |
| 经济逻辑 | "我"的经济状况和角色匹配吗？（被养的人不可能有公司、不需要应酬客户） |
| 关系逻辑 | "我"和对方的关系匹配吗？（被养的人不可能给家里钱、不需要租房） |
| 身份逻辑 | 对方的身份和新角色匹配吗？（富婆不可能在柜台兼职、不可能租破房子） |
| 情节逻辑 | 这个情节在新关系中说得通吗？ |

## "我"的人物设定

### 软饭版（默认人设）
- 身份：上门女婿，全职带娃
- 老婆：做金融/家里做生意，年薪高
- 经济：没有收入，吃住都是老婆家给
- 日常：男保姆，负责做饭、带孩子、做家务
- 住：住在老婆家/岳父家
- 零花钱：老婆每月给2000-5000
- 社交：朋友问就说"创业/自由职业"，没人信
- 心态：怕老婆说"你出去找个工作吧"

### 包养版（反向改写时的人设）
- 身份：被包养的人
- 对方：有钱，养我
- 经济：没有收入，花对方的钱
- 日常：在对方那里做饭、等对方、陪对方
- 住：对方提供住处
- 零花钱：对方给
- 心态：怕对方说"我们不能再见了"

### 富婆版（老婆是真富的人设）
- 身份：被富婆养的人
- 老婆：真富婆，存款A8-A9级别（千万-上亿）
- 资产管理：信托账户、理财、投资，不是"卡里"
- 住：老婆的房子（婚前买的，写她名字）
- 零花钱：不写具体数字，用"一笔钱""不少"等模糊词
- 抚养费：如果离婚，用"天价抚养费"，不写具体数字
- 地理细节：要具体（比如"深圳关外城中村"）
- 心态：离婚了什么都没有，连住的地方都没有

## 双向改写公式

### 方向A：包养素材 → 软饭文学

| 原素材元素 | 改写后元素 |
|-----------|-----------|
| 情人/被包养的人 | 老婆（金融/高管/家里做生意） |
| 我给情人租房 | 我住在老婆家/岳父家 |
| 每月给情人钱 | 老婆每月给我零花钱 |
| 偷偷见面 | 住在一起 |
| 分手费 | 离婚后我什么都没有 |
| 两个微信隐藏 | 朋友问就说"创业/自由职业" |
| 怕被老婆发现 | 怕被丈母娘挑剔 |
| 我有工作/有钱 | 我没有收入，全靠老婆 |

### 方向B：软饭素材 → 包养文学

| 原素材元素 | 改写后元素 |
|-----------|-----------|
| 老婆养我 | 对方包养我 |
| 住在老婆家 | 对方给我租房/住在对方那里 |
| 老婆给我零花钱 | 对方给我生活费 |
| 怕老婆让我出去工作 | 怕对方说"我们不能再见了" |
| 朋友问说"创业" | 朋友不知道我被包养 |
| 我没有收入 | 我没有收入（不变） |
| 饭是我做，孩子是我接 | 饭是我做（不变） |
| 散了我什么都没有 | 散了我什么都没有（不变） |

**关键：方向B中，"我"的角色不变——还是被养的那个人。只是关系从"夫妻"变成"包养"。**

## 输出格式

直接输出推文正文，不需要"关于xx"的格式（除非用户要求）。

## 去 AI 味规则

- 禁用"本质上"、"事实上"、"实际上"
- 禁用"不是...而是..."句式
- 禁用"不仅...而且..."句式
- 禁止金句式结尾
- 口语化，像唠家常
- 短句独立成段，用换行代替句号
- 标题和结尾去掉句号
- 避免太具体的小数字（用"天价""一大笔""不少"等模糊词）
- 用模糊词代替具体存款数字（"有的是钱""不差钱" > "120万存款"）
- 用口语化的模糊词（"乱七八糟的支出" > "房租、吃饭、交通"）
- 增加真实生活细节（刷抖音、看网红、听别人说）
- 软饭人设的核心恐惧：不是怕钱没了，是怕挨骂/被赶出去
- 焦点在"我"，不写老婆的具体收入
- 包养关系的消费水平要匹配（298 → 2980）
- 有钱人不说"太贵"，说"不健康""不放心""不合口味"
- 避免地理细节重复，换不同城市
- 包养关系用隐晦词（"我男朋友" → "家里那谁"）
- **禁止提到"当兵的"、"军人"等军事相关职业**

## 软饭文学核心特征（2026-03-03更新）

**软饭文学是写经历，不是写观点。**

### 十大核心特征

| 特征 | 说明 | 案例 |
|------|------|------|
| 1. 叙事性 | 是讲故事，不是讲道理/观点 | "第二个念头是找工作"讲的是算账放弃的过程 |
| 2. 具体到极致 | 数字、时间、地点都要具体 | 2980一节课、135一斤、2022年10月、深圳关外城中村 |
| 3. 算账 | 总是算钱、算收益、算成本 | 开奶茶店要40万、4年回本、月入2.3万扣掉支出剩不了多少 |
| 4. 时间线 | 有明确的时间推进 | 11月、今年3月、去年6月、第一次、后来 |
| 5. 核心恐惧 | 怕挨骂、怕被说"出去找工作"、怕被赶出去 | "钱没了无所谓，但我绝对要挨她一顿骂" |
| 6. 结尾无力 | "算了"、"该做饭了"、"土豆切完了" | 挣扎之后回归现实，接受现状 |
| 7. 软饭男视角 | 被养的姿态，想逃但逃不掉的纠结 | 存了半年也就1万、想给她转2万但没钱 |
| 8. 碎碎叨叨 | 短句、换行、像在跟朋友聊天 | 没有分点，没有"关于XX"的格式 |
| 9. 市井生活 | 学校门口、便利店、菜市场、奶茶店、糖炒栗子店 | 学校门口新开了一家糖炒栗子店 |
| 10. 情感克制 | 不说"爱"，不说"恨"，说"算了" | 没哭，就是站着；只是有时候会想起 |

### 适配性判断新增

❌ **不适合改写的素材类型（新增）**：
- 抽象观点类：如"挑老婆要看三点"、"如何选择伴侣"
- 干货分享类：没有具体情节、时间线、细节的内容

✅ **适合改写的素材类型（新增）**：
- 有具体情节的故事：可以有时间推进、有细节描写
- 有算账过程：能展示软饭男的纠结
- 有情感挣扎：想逃但逃不掉的张力

### 典型案例对比

| 类型 | 示例 | 是否软饭文学 | 原因 |
|------|------|-------------|------|
| 软饭文学 | "第二个念头是找工作" | ✓ | 有故事、有算账、有细节、有挣扎 |
| 软饭文学 | "学校门口新开了一家糖炒栗子店" | ✓ | 有时间线、有情节、有具体数字 |
| 非软饭文学 | "挑女婿要看认知格局情绪" | ✗ | 是观点，不是故事，没有细节 |
| 非软饭文学 | "如何选择老婆的三点建议" | ✗ | 是干货，没有情节和挣扎 |

## 人机协作迭代机制

### 工作流程

**Step 1：AI改写**
- 按照现有规则改写素材
- 输出初稿

**Step 2：人工修改**
- 用户用人的视角再改一遍
- 标注修改的地方

**Step 3：AI分析**
- 我分析用户为什么这么改
- 识别改动的模式和原因
- 总结成可复用的规则

**Step 4：规则更新**
- 把新发现的规则更新到子技能文件
- 分类存档：
  - 事实逻辑问题
  - 语言表达问题
  - 情节细节问题
  - 情感节奏问题

**Step 5：下次改写时应用**
- 新规则自动生效
- 持续优化

### 分析框架

当用户提供修改后的版本时，我会从以下维度分析：

| 维度 | 分析内容 |
|------|---------|
| 事实逻辑 | 是否修正了性别、经济、关系、身份的逻辑错误？ |
| 语言表达 | 是否改得更口语化、更自然、更像人说话？ |
| 情节细节 | 是否补充/删除了某些细节？为什么？ |

## 迭代记录

### 迭代1 - 2026-03-02

**素材类型**：包养素材 → 软饭文学

**原文核心**：
- 我（有钱）包养女大学生，想离婚但算了算账放弃了

**AI初稿问题**：
1. 语言节奏太慢：大段落，句号多，不像口语
2. 存款金额不匹配：写"80多万"，但富婆应该是真富（A8-A9级别）
3. 资产管理方式不真实：写"都在她卡里"，富人应该用信托
4. 抚养费金额不合理：写"2000抚养费"，富婆的抚养费不可能这么少
5. 地理细节不具体：写"租房"，应该写具体地点
6. 情感层次不够：缺少呼应句和强调句
7. 太确定：写"一人一半，我能分40万"，应该更不确定

**人工修改**：
1. 分段更碎：短句独立成段，用换行代替句号
2. 存款改成"A9"：上亿级别，符合富婆人设
3. "卡里"改成"信托账户里"：更符合富人资产管理方式
4. "2000抚养费"改成"天价抚养费"：避免具体小数字
5. "租房"改成"深圳关外的城中村"：具体地理细节
6. 增加"我确实跟富婆一起了"：呼应前面的问题
7. "一人一半，我能分40万"改成"我都不知道能不能一人一半"：更不确定
8. 增加"但是离婚了就什么都没了"：强化绝望感
9. 删除"每个月她给我生活费3000"：太具体反而不真实

**提炼的新规则**：

**规则1：语言节奏要碎**
- 短句独立成段，用换行代替句号
- 标题和结尾去掉句号
- 节奏要快，像口语

**规则2：富婆人设要匹配**
- 存款：A8-A9级别（千万-上亿），不是几十万
- 资产管理：信托、理财、投资，不是"卡里"
- 抚养费：用"天价"，不要写具体小数字

**规则3：地理细节要具体**
- "租房" → "深圳关外城中村"
- "住哪" → 具体的地名/区域

**规则4：情感层次要递进**
- 增加呼应句（让情节更连贯）
- 用不确定感代替确定感（"我都不知道" > "我能分"）
- 增加强调句（放大情绪）

**规则5：避免太具体的小数字**
- 不要写"生活费3000""抚养费2000"（容易露馅）
- 用"天价""一大笔""不少"等模糊词

**应用到**：
- 去AI味规则（新增3条）
- 人物设定（富婆版）
- 改写公式（软饭版）

### 迭代2 - 2026-03-02

**素材类型**：包养素材 → 软饭文学（想辞职创业）

**原文核心**：
- 我（有工作有收入）想辞职创业，算了算账后放弃了

**AI初稿问题**：
1. 写了具体存款数字："80多万存款，再加上这些年攒的，大概120万左右"
2. 缺少真实生活细节：没有刷抖音、看网红等真实场景
3. 恐惧点不对：写"万一做不起来，40万就没了"，但软饭人设应该怕挨骂
4. 写了老婆的具体收入："老婆每月给我5000零花钱"
5. 列举太具体："扣掉房租、吃饭、交通"

**人工修改**：
1. "120万存款"改成"有的是钱"：更模糊，更口语化
2. 增加"每天刷抖音看到勇哥说餐饮那么多人都被割韭菜"：真实生活细节
3. "万一做不起来，40万就没了"改成"钱没了无所谓，但是我绝对要挨她，和她爸妈一顿骂"：强化软饭人设的核心恐惧
4. 删除老婆的具体收入描述
5. "扣掉房租、吃饭、交通"改成"扣掉乱七八糟的支出"：更口语化

**提炼的新规则**：

**规则6：用模糊词代替具体存款数字**
- "120万存款" → "有的是钱"
- "80万" → "不差钱"

**规则7：增加真实生活细节**
- 刷抖音、看网红、听别人说
- 增加真实感和生活化

**规则8：软饭人设的核心恐惧**
- 不是怕钱没了，是怕挨骂
- 不是怕失败，是怕被赶出去
- 权力关系：怕老婆、怕岳父母

**规则9：焦点在"我"，不写老婆的收入**
- 不需要写老婆月收入多少
- 焦点在"我"的感受和处境

**规则10：用口语化的模糊词**
- "扣掉房租、吃饭、交通" → "扣掉乱七八糟的支出"
- 更自然，更像人说话

**应用到**：
- 去AI味规则（新增5条）

### 迭代3 - 2026-03-02

**素材类型**：软饭素材 → 包养文学（学做饭遇到林老师）

3. 地理细节重复：用了"深圳"，和前面的"深圳关外城中村"重复
4. 关系称呼太直接：写"我男朋友做饭可好吃了"，包养关系不会公开说

- "外面吃太贵了" → "外面吃不健康了"

- 保持地理细节的多样性

**规则14：包养关系用隐晦词**
