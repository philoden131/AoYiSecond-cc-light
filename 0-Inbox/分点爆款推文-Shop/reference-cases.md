# 参考案例库

## 高表现案例（10K+ 印象）

### 案例 1：人性/社会 × 观察/发现型开头
```
发现一个规律，经济越差，某些生意反而越好：

算命的、卖焦虑课的、搞灵修的

因为人在顺境时相信自己
逆境时开始相信命运

这不是迷信复兴
这是集体安全感崩塌后的精神消费升级
```
分析：观察/发现型开头 + 反常识 + 结尾点破本质

### 案例 2：认知/成长 × 话题直入型
```
东亚家庭最大的矛盾：

小时候打压你，长大了叫你自信点
小时候替你做决定，长大了怪你没主见
小时候不让你哭，长大了说你冷漠

这是他们在不同阶段对你有不同的用处罢了
```
分析：话题直入 + 每点都有前后矛盾冲突 + 结尾一刀见血

### 案例 3：商业/经济 × 第一人称
```
我花了一周时间，把主流AI的Pro版全买了一遍

结论：
90%的人根本不需要付费版
免费版够用，省下的钱买本书更值

真正需要Pro的只有两种人：
靠AI吃饭的，和用AI省时间换钱的
```
分析：第一人称+具体行动+信息差，结尾给出实用判断

## 失败案例（<2K 印象）

### 失败 1：鸡汤型
```
真正独立的人都有这些特质：

自己赚钱养活自己，不靠父母也不靠伴侣
做决定的时候不会因为别人怎么说就改主意
有爱情亲情友情当然好，但没有也不会慌

大概是因为他们不需要靠关系来获得安全感吧
```
失败原因：列点没有冲突/对比，只是平铺特质；结尾用"大概"太软；读完只想点头不想转发

### 失败 2：知识科普型
```
AI Agent 和传统 AI 的区别：

传统AI：你问它答，一问一答
Agent：自己规划、自己执行、自己纠错

简单说就是从"工具"变成了"员工"
```
失败原因：纯知识科普，没有情绪触发；没有和读者生活产生关联；信息量不够独家
