# 标签词汇表

> 标签是跨领域的桥梁。新增前先查此表，优先复用已有标签。
> 结构标签 = type 字段值，受控，新增需 B 类审批。
> 主题标签 = tags 字段值，半开放，Claude 负责去重。
> 由 build_pointers.py 自动统计使用计数，Claude 负责描述和去重。

## 结构标签

| 标签 | 含义 |
|------|------|
| 方法论 | 系统性的做事方法 |
| 笔记 | 学习心得、课程记录、信息摘录 |
| 决策 | 选择与理由记录 |
| 案例 | 案例拆解与分析 |
| 提示词 | 可复用的 AI 提示词模板（含系统提示词） |
| 价值观 | 指导原则与信念 |
| 体系 | 多方法论组成的完整框架 |
| 设定 | 角色/环境/规范的配置定义 |
| 作品 | 最终交付物：文案、视频脚本、发布稿 |
| 系统文档 | 系统运行规范、身份定义等基础设施文件 |

> 废弃映射：沉淀→笔记, 临时草稿→笔记+status:draft, 草稿→笔记+status:draft, 系统提示词→提示词, 文案→作品, 成品→作品, 文档→作品
> 入库时优先复用已有类型；确需新增时添加到此表并写清含义。
