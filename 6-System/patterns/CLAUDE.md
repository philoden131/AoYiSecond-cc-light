# 6-System/patterns/
> L2 | 父级: /6-System/CLAUDE.md

## 职责

任务智能积累：存放成功的任务执行模式，可复用的交互策略。

**这是"好用"轨道**——每次成功的复杂交互沉淀在此，下次同类任务直接加载，不从零摸索。

## 什么时候写入

- 复杂任务成功完成后，AI 识别到有价值的执行方法
- 用户明确说"这次做法很好，记下来"
- /digest 触发时，AI 从对话中识别可复用模式

## 文件命名

`[任务类型]-[具体方法].md`

例：
- `writing-口播稿创作流程.md`
- `research-竞品分析框架.md`
- `system-知识库入库流程.md`

## 写入格式

```markdown
# [模式名称]

**适用场景**：[什么情况下用这个模式]
**成功案例**：[来自哪次对话，日期]

## 执行步骤
1. ...
2. ...

## 关键经验
- [这次成功的核心原因]
```

## 现有模式

- `system-absorb技能设计考量.md`: /absorb 统一内化技能的设计考量，待多次场景验证后实现
- `tasks-system-design.md`: 任务捕获/追踪/归档完整系统设计方案，含文件清单、流程说明、用户路径走查（待实施）
