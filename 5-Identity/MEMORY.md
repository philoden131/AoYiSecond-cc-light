# 身份记忆主入口

> 这是「了解奥一是谁」的检索起点。
> 涉及用户偏好、长期行为模式、价值观、AI 行为规则时，**优先读此目录下的文件**，不要去 `6-System/working-memory/`。

---

## 检索优先级

| 文件 | 内容 | 何时读 |
|------|------|--------|
| `TELOS.md` | 使命/目标/信念/策略/框架 | 做价值判断、战略决策时 |
| `AI_RULES.md` | AI 行为准则与角色定义 | 理解如何服务奥一时 |
| `CONTEXT.md` | 当前动态上下文（活跃项目/近期关注/近期决策） | 了解奥一现在在做什么时 |
| `PROFILE.md` | 场景化偏好（交互/写作/执行/学习） | 调整回复风格和方案偏好时 |

---

## 与系统工作记忆的区别

| | `5-Identity/`（此处） | `6-System/working-memory/` |
|---|---|---|
| **语义** | 用户记忆主入口 | 系统工作记忆 |
| **内容** | 奥一是谁、偏好、行为模式 | 执行约定、流程规范、操作性规则 |
| **查询场景** | 「奥一喜欢什么？」「他的目标是？」 | 「文件命名怎么写？」「审批流程是？」 |
